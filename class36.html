Javascript IF-ELSE Statements

The Javascript if else statement is used to execute the code
whether the condition is true or false
There are 2 forms of if statements in Javascript
1. If statement
2. If else statement
IF evaluates the content only if expression or condition is true
if(condition){ code to be executed if it is true }
IF ELSE evaluates the content when the conidition is true or false with an extra else statement
if (){ } else { }
